포트폴리오 편집기

코드를 직접 만지지 않고 작업물·소개를 추가할 수 있어요. ① web 폴더 열기 → ② 편집 → ③ 저장(폴더에 바로 기록).

① 사이트 폴더 열기

index.html이 들어 있는 web 폴더를 선택하세요. 이미지는 폴더 안 images/에 자동 저장됩니다. (Chrome 또는 Edge에서 열어주세요.)

② 작업 (연도별 프로젝트)

연도 안에 프로젝트를 만들고, 프로젝트 안에 이미지를 넣으세요. 이미지를 고르면 파일 안에 바로 포함됩니다(별도 폴더 불필요).

② Information (소개·이력·연락처)

③ 저장

저장을 누르면 폴더 안의 index.html이 바로 갱신되고, 새로 올린 이미지는 images/에 기록됩니다. 그다음 브라우저에서 index.html을 새로고침하면 끝이에요.

편집 내용은 "저장"을 눌러야 폴더에 기록됩니다.
